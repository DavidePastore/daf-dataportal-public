# data-portal

[![Build Status][ico-travis]][link-travis]
[![Coverage][ico-codecov]][link-codecov]

React/Redux data-portal skeleton

Start in Debug Mode:
```
npm install
npm start
```

Start with mock server:
```
npm run mock
```

Start in Production Mode:
```
npm run build
npm install -g serve
serve -s build
```
